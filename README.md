## Descargar repo
```powershell
git clone https://github.com/janselroa/web-clima
```

## Instalar Dependencias

```
pip install -r requirements.txt
```

## Arrancar App
### powershell

```
$env:FLASK_APP = "main"
flask run
```
### cmd

```
set FLASK_APP=hello
flask run
```

### bash

```
export FLASK_APP=hello
flask run
```
